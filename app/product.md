Overview
RewriteMate is an AI keyboard extension for iOS and macOS that lets users rewrite, rephrase, grammar-check, translate, and generate text directly inside any app — without copy-pasting or switching contexts. It features over 14 built-in commands (like Proofread, Paraphrase, and Change Tone) and allows users to create fully custom commands with personalized icons and colors.
The product features two main areas:
Module	Role	User Experience
Main App	Management Hub	Used for onboarding, managing custom commands/folders, selecting AI models, and managing subscriptions.
Keyboard Extension	The Writing Surface	The actual keyboard that works seamlessly inside any text field across any app.
Key Differentiator: Unlike standalone AI apps that require copying and pasting text back and forth, RewriteMate lives directly inside the system keyboard. The rewriting happens instantly right where the user is already typing.

Main Features & UI Context
1. One-Tap Rewriting Commands
• What it is: Over 14 built-in AI commands accessible directly from the keyboard's top bar.
• UI Experience: The user selects text and taps a command chip (e.g., Proofread, Paraphrase, Shorten) on the keyboard. The AI processes the text and streams the result into an overlay above the keys.
Three immediate actions are available:
• Copy: Save the full response to the system clipboard.
• Insert: Place the response at the current cursor position.
• Replace: Automatically swap out the selected text or sentence with the new AI version.
2. Custom Commands
• What it is: The ability for users to build their own personalized rewrite shortcuts.
• UI Experience: Created in the main app using a simple creation form where users choose a custom icon, brand color, title, and specific AI prompt instructions. These appear as colorful cards in the app and as quick-access chips inside the keyboard.
3. Folder Organization
• What it is: Grouping commands into categories for faster navigation.
• UI Experience: Users can organize their custom and built-in commands into folders (like Work, Social, Replies) within the main app. In the keyboard, these folders appear as pill-shaped filter tabs right above the command grid.
4. Compose (AI Chat)
• What it is: A free-form text input for typing direct, unique instructions to the AI.
• UI Experience: A dedicated tab featuring an interactive text field. Users can type specific requests like "Make this sound more urgent" or "Rewrite this as a bulleted list", combining their instructions with the text currently on screen.
5. Instant Translation
• What it is: Translating text into over 100 languages without leaving the app you are typing in.
• UI Experience: A translation tab marked with a globe icon displays a quick-access row of language options. Tapping a language instantly translates the selected text.
6. Writing History
• What it is: A quick-access archive of previously used commands.
• UI Experience: A dedicated clock icon tab displaying a grid of recently used actions so users can re-run frequent styles without searching for them.
7. Multi-Model AI Selection
• What it is: Freedom to choose which foundational AI intelligence powers the rewrites.
• UI Experience: Accessible in the profile tab. Premium users can toggle between leading AI models from OpenAI (GPT), Anthropic (Claude), and Google (Gemini) based on their personal text preferences.
8. Prompt Playground
• What it is: A testing sandbox for custom prompts.
• UI Experience: Located inside the custom command creation screen. It lets users test out how their custom instructions behave with sample text before saving the shortcut to their keyboard.
